# Z_Fortune_Cookie

A simple Python script meant for learning purposes that emulates a fortune cookie

---

## Features

* Greeting the user
* Handling user input
* Displaying a fortune to the console

---

## Installation

Clone the repository:
```bash
git clone https://github.com/Zeno-Liber/Z_Fortune_Cookie.git
cd Z_Fortune_Cookie
```
---

## Usage

Run the program:

```bash
z_fortune_script.py
```

This program will greet the user then prompt to select an option. Then will display a fortune to the console.

## Requirements

* Python 3.12
